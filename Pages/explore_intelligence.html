5 Core Intelligence Modules

Each module specializes in a critical aspect of learning intelligence

üéØ

Focus Detection Engine

Continuously monitors attention quality through eye gaze patterns, facial micro-expressions, and interaction signals.

How It Works
Computer Vision Pipeline:
  • Eye tracking via pupil detection (60 FPS)
  • Gaze direction mapping to screen regions
  • Blink rate analysis for fatigue detection
  • Head pose estimation for engagement
  • Focus score calculated every 500ms
Models Used: MediaPipe Face Mesh, OpenCV Eye Tracking, Custom CNN for attention classification
üòä

Emotional Intelligence Layer

Detects 7 core emotions and responds with appropriate teaching adjustments and empathetic interventions.

How It Works
Facial Emotion Recognition:
  • 68-point facial landmark detection
  • Action Unit (AU) classification
  • 7 emotions: joy, confusion, frustration, boredom, excitement, anxiety, neutral
  • Temporal smoothing for stable predictions
  • Context-aware emotion understanding
Models Used: FER2013-trained ResNet, EfficientNet for real-time inference, LSTM for temporal context
üß†

Adaptive Learning Algorithm

Dynamically adjusts content, difficulty, and pacing based on real-time understanding and historical performance.

How It Works
Reinforcement Learning System:
  • State: Current knowledge, emotion, focus, context
  • Actions: 47 personalization parameter adjustments
  • Reward: Learning efficiency, mastery achievement
  • Policy trained on 10M+ student sessions
  • A/B tested continuously for optimization
Algorithm: Deep Q-Network (DQN) with experience replay, PPO for policy optimization
üìä

Learning Impact Index (LII) Calculator

Comprehensive scoring system that captures true learning progress across 5 dimensions beyond test scores.

How It Works
Multi-Dimensional Scoring:
  • Focus Quality (20%): Avg focus score, consistency
  • Mastery Achievement (30%): Concept understanding depth
  • Curiosity Index (15%): Questions asked, exploration
  • Resilience Score (15%): Persistence through difficulty
  • Consistency (20%): Regular engagement, habits
Calculation: Weighted ensemble of normalized sub-scores, updated in real-time
üí°

Curiosity Measurement System

Tracks and encourages intellectual curiosity through question analysis, exploration patterns, and deep thinking.

How It Works
Curiosity Indicators:
  • Question quality analysis via NLP
  • "Why" and "how" question frequency
  • Off-syllabus topic exploration
  • Deep dive duration into concepts
  • Connection-making between topics
Models Used: BERT for question classification, Graph Neural Networks for concept connections
